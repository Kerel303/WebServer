Java ServerSocket

Podstawy komunikacji sieciowej w standardzie TCP/IP

Czym jest ServerSocket?

Klasa ServerSocket w języku Java służy do tworzenia programów serwerowych. Oczekuje ona na przychodzące połączenia sieciowe od klientów (za pomocą standardowych protokołów TCP) na określonym porcie, a po zaakceptowaniu połączenia umożliwia dwukierunkową wymianę danych.

Kluczowe kroki działania

  • Inicjalizacja: Utworzenie obiektu i przypisanie go do wolnego portu.
  • Oczekiwanie: Wywołanie metody accept(), która blokuje program do momentu połączenia klienta.
  • Strumienie danych: Pobranie strumieni wejścia/wyjścia do komunikacji.
  • Zamknięcie: Zwolnienie zasobów po zakończeniu transmisji.

Przykładowy Kod Serwera

Poniższy kod przedstawia prosty serwer Echo, który odbiera tekst od klienta i odsyła go z powrotem:

import java.io.*;
import java.net.*;

public class ProstySerwer {
    public static void main(String[] args) {
        int port = 5000;
        
        try (ServerSocket serverSocket = new ServerSocket(port)) {
            System.out.println("Serwer uruchomiony na porcie " + port);
            
            // Oczekiwanie na klienta
            Socket clientSocket = serverSocket.accept();
            System.out.println("Połączono z klientem!");
            
            // Tworzenie strumieni komunikacji
            BufferedReader in = new BufferedReader(new InputStreamReader(clientSocket.getInputStream()));
            PrintWriter out = new PrintWriter(clientSocket.getOutputStream(), true);
            
            // Odczyt i odesłanie wiadomości
            String inputLine = in.readLine();
            out.println("Serwer odpowiada: " + inputLine);
            
        } catch (IOException e) {
            System.out.println("Błąd serwera: " + e.getMessage());
        }
    }
}
